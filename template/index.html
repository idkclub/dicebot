Clicking the button above will add the following command to your team's Slack:

/roll [XdY] Roll X Y-sided dice

which will allow members to roll dice using Standard Notation.

For example, /roll 2d6 will simulate rolling 2 6-sided dice, and return the sum via an in-channel post by Dicebot.

The full set of rolling options, along with some example commands for different gaming systems, is explained in the DiceBot README.
